# Max Pain 趋势跟踪策略规则

**更新**：2026-06-24 (v5 纯MP)
**品种**：PTA (TA)、MA (甲醇)、SA (纯碱)
**合约乘数**：5 / 10 / 20
**数据范围**：2020~2026 年 ZCE 期权日频数据

---

## 1. 策略架构

```
MP趋势方向 → 斜率自适应追踪止盈 + 保本锁 + 方向反转出场 + 连胜跳过
```

纯 MP（Max Pain）趋势跟踪。无额外指标，无市场状态分类，无复杂出场结构。核心逻辑：MP 方向代表期权市场的"最痛"位置漂移方向，跟随这个方向交易。

---

## 2. 开仓逻辑

### 2.1 MP 趋势方向 — 21 日线性回归斜率

```
slope     = (n·Σxy - Σx·Σy) / (n·Σx² - (Σx)²)
projected = slope × (n-1)          // 窗口总变化
threshold = 窗口起始值 × Min%(1%)
```

| projected 与 threshold | 判定 |
|------------------------|------|
| `> threshold` | up（上涨） |
| `< -threshold` | down（下跌） |
| 其余 | null（无方向） |

### 2.2 入场条件

```
MP趋势 up   + prevClose > prevMp  → 开多
MP趋势 down + prevClose < prevMp  → 开空
```

- 使用前日收盘价（`prevClose`）和前日 MP（`prevMp`）避免预知未来
- 价格必须在 MP 同侧：做多时前收盘在 MP 之上，做空时在前收盘 MP 之下

### 2.3 连胜跳过

某方向盈利出场后，跳过后 N 次反方向信号。

| 场景 | 行为 |
|------|------|
| 开多盈利 → | 跳过后面 N 次做空信号 |
| 开空盈利 → | 跳过后面 N 次做多信号 |

N 默认 2，可配置（0=关闭）。用意：顺方向赚到后暂停反方向开仓，避免反向信号频繁磨损。

## 3. 出场规则

### 优先级 1：方向反转出场

| 方向 | 出场条件 |
|------|---------|
| 多头 | MP 趋势由 `up` 变为 `down` 或 `null` |
| 空头 | MP 趋势由 `down` 变为 `up` 或 `null` |

趋势不存在时持仓逻辑失效。出场后同一根 K 线可反向开仓。

### 优先级 2：MP 斜率自适应追踪止盈 + 保本锁

```
MP投影 = linearProjected(mpHist, LBK)
MP斜率 = |MP投影| / |prevMp|
trail  = max(Min%, 保本%) × (1 + 斜率 / 0.05)
```

斜率趋近 0 → trail ≈ 1%。斜率越大（趋势越强），trail 越宽防止被洗：

| MP斜率 | trail 距离 |
|--------|-----------|
| 0%     | 1.0% |
| 5%     | 2.0% |
| 10%    | 3.0% |
| 15%    | 4.0% |

**保本锁定**：价格朝有利方向移动超过 `保本%` 后激活。锁定 `峰值利润 × 锁定%` 作为最低止盈。

**多头止损计算**：
- `stop = prevClose × (1 - trail)`
- 若已激活保本锁：`stop = max(stop, entryPx + (peakPx - entryPx) × 锁定%)`
- `lo < stop` → 出场

**空头止损计算**：
- `stop = prevClose × (1 + trail)`
- 若已激活保本锁：`stop = min(stop, entryPx - (entryPx - peakPx) × 锁定%)`
- `hi > stop` → 出场

## 4. 参数表

| 参数 | 默认 | 优化范围 | 说明 |
|------|------|---------|------|
| Lookback | 21 | 5~60 | 趋势计算窗口 |
| Min% | 1.0% | 0.2~5% | MP 趋势分类阈值 |
| 仓位% | 30% | 5~60% | 单笔最大资金占用 |
| 保证金% | 15% | — | 交易所标准 |
| 保本% | 1.0% | 0~5% | 激活利润锁定所需盈利 |
| 锁定% | 75% | 10~70% | 锁定峰值利润比例 |
| 跳过次 | 2 | 0~5 | 盈利后跳过同向信号次数 |
| 本金 | 100,000 | — | 初始权益 |
| 封顶 | 0 | — | 单笔名义本金上限，0=不限 |

## 5. 设计演进

| 阶段 | 方向信号 | 过滤 | 出场 | 备注 |
|------|---------|------|------|------|
| 初始 | 3信号加权(VR+IVS+BE) | — | 多条件 | 过拟合 |
| 简化 | MP趋势斜率 | OI偏度确信度 | trail+结构击穿 | — |
| v1 | MP趋势斜率 | 发散门+IVS确信度 | trail+锁定+结构击穿 | — |
| v2 | MP趋势斜率 | 发散门 | trail+锁定+结构击穿+方向反转 | — |
| v3 | MP趋势+R² | 趋势MP附近/震荡BE附近 | 结构+痛点止盈+方向反转+trail | 自适应双模式 |
| v4 | MP趋势+ADX | ADX筛状态+RSI筛入场 | 结构+痛点止盈+方向反转+ATR | ADX+RSI+ATR |
| **v5** | **MP趋势** | **价格同侧** | **方向反转+斜率自适应trail+保本锁+连胜跳过** | **纯MP** |

**关键迭代**：
- v4→v5：去掉 ADX/RSI/ATR，回归纯 MP
- 斜率自适应替代固定 trail：趋势强时放宽防止被洗
- 连胜跳过：盈利后暂停同向信号，避免追高杀跌
- 结构击穿改为仅显示，不参与出场

## 6. 策略特征

| 方面 | 说明 |
|------|------|
| **风格** | MP 趋势跟踪 |
| **持仓周期** | 5~15 天 |
| **核心逻辑** | MP 漂移方向即趋势方向，跟随交易 |
| **优势** | 极简，参数少，过拟合风险低 |
| **风险** | MP 在震荡市中频繁变向导致磨损；趋势快速反转时方向反转出场可能滞后 |
